## Description
This is my Final Project for **Harvard University's CS50P**. It is a Python-based application designed to [MÔ TẢ NGẮN GỌN DỰ ÁN LÀM GÌ, VÍ DỤ: fetch real-time weather data and save it to a CSV file].

### Features
* **Data Retrieval**: Uses the `requests` library to interact with external APIs.
* **Input Validation**: Ensures user input is correct using Python's built-in string methods and regular expressions (`re` module).
* **Automated Testing**: Includes a `test_project.py` file with multiple `pytest` functions to ensure code reliability.

### Project Structure
* `project.py`: The main entry point of the application containing the core logic.
* `test_project.py`: Contains test cases to be run with `pytest`.
* `requirements.txt`: Lists all external libraries (like `requests` or `pyfiglet`) needed to run the app.
